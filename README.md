### 🚀 Running the project

#### Clone this repository :

```javascript
git clone https://github.com/Pauline-Bouchereau/job-board
cd job-board
```

#### Install packages :

```javascript
yarn
```

#### When installation is complete :

```javascript
yarn start
```
